# Food Product Explorer

A modern and premium food product discovery web application built using **React**, **Vite**, and **Tailwind CSS**, powered by the **Open Food Facts API**.

This project allows users to search food products, explore nutritional information, analyze ingredients, compare Nutri-Scores, and view detailed product insights through a clean, minimal, and professional UI.

Designed with a strong focus on **UI/UX**, responsiveness, and production-level frontend practices, this project is ideal for portfolio showcase, internships, and real-world frontend development demonstrations.

---

# Features

## Home Page

* Premium modern landing section
* Product search functionality
* Category-based filtering
* Product sorting options
* Real-time product fetching using Open Food Facts API
* Responsive product grid
* Pagination support
* Loading states
* Error handling
* Empty state handling

## Product Details Page

* Detailed product information
* Product image preview
* Brand and category display
* Nutrition overview (Energy, Protein, Fat, Carbs, Sugar, Salt)
* Ingredients section
* Labels and allergens
* Nutri-Score display
* Product metadata (Barcode, Quantity, Serving Size)

## UI/UX

* Premium dark minimal design
* Clean typography and spacing
* Responsive layout for all devices
* Modern professional color palette
* Minimal and polished interface
* Production-ready structure

# Installation

## Clone the repository

```bash
git clone https://github.com/your-username/food-product-explorer.git
cd food-product-explorer
```

## Install dependencies

```bash
npm install
```

## Run development server

```bash
npm run dev
```

## Open browser

```bash
http://localhost:5173
```

(Vite may automatically use another port if 5173 is occupied)
